## 题型

- 链表
  - 反转
  - 快慢指针
- 数组
  - 双指针
  - 滑动窗口
  - 求和，消失的数字
  - 二分查找
  - 快速排序
  - 子区间，前缀和
- 栈
  - 计算器、运算符、括号匹配
  - 单调栈
- 队列
  - 双栈实现队列
  - 单调队列
  - 优先队列（大顶堆）
- 树、图
  - 二叉树遍历、镜像、最大，小深度、二叉树重建
  - 广度优先遍历
  - 深度优先遍历
  - 并查集
  - 递归、回溯、剪枝（去重）
- 哈希表
- 排序
  - 拓扑排序
  - 快速排序
- 动态规划
  - 背包
  - 爬楼地
  - 子区间最大值
- 数学规律
  - 下一个排列：从后往前找到第一个上坡，k < k+1；再从后向前找到第一个大于k的，然后交换
  - 最大数：两个数字字符串相加后，大的那一个在前面
